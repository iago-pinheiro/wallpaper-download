# 🖼️ Wallpaper Corporativo - CDN

**Descrição:** Repositório dedicado à hospedagem estática e centralizada dos wallpapers corporativos da empresa via GitHub Pages. 

Este repositório atua como um CDN (Content Delivery Network), garantindo que os papéis de parede sejam distribuídos rapidamente e sem limite de banda para todos os notebooks dos colaboradores (através do nosso futuro script de atualização).

---

## 🚀 Como atualizar o Wallpaper

Para alterar o papel de parede de toda a empresa de forma imediata e transparente:

1. A imagem sempre deve ter o nome exato: **`wallpaper.jpg`**.
2. É recomendável usar o formato **JPG/JPEG** para equilibrar boa qualidade e tamanho leve.
3. No GitHub, clique em **"Add file"** > **"Upload files"**.
4. Arraste a nova imagem `wallpaper.jpg` substituindo a antiga.
5. Clique verde no botão de **"Commit changes"**.

> **Pronto!** Em 1 minuto o link do GitHub Pages vai propagar a nova imagem pelo mundo, e o sistema cliente instalado nos computadores vai buscar essa nova versão no próximo login do usuário.
